Instructions/Requirements

First, download the image by right-clicking it and selecting “Save Picture As,” then save it to your folder. Next, create a blank web page and save it as img_map.html. Insert the image using the tag and include the width (600) and height (396). Open the page in a browser to check if it works. Then, use a graphics tool like Microsoft Paint or Image Map Generator to find the coordinates of the parts you want to make clickable. After that, create an image map with four clickable areas based on the numbered buildings. Add the correct links and alt text for each area. Finally, submit and upload your file to your portfolio.
